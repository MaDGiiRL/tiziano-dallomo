# Tiziano Dall’Omo SPA

Versione aggiornata con:

- palette grigio e blu
- contatti reali
- testi ricavati dal materiale fornito
- React + Vite
- Tailwind CSS v4 senza PostCSS
- sezioni dinamiche con slider, paginazione e tab

## Avvio

```bash
npm install
npm run dev
```
